# 欢迎使用知识库

这是你的 Obsidian 知识库根目录。记录内容！

## 目录说明

| 目录 | 用途 |
|------|------|
| `00-收件箱` | 临时存放，每日清理归类 |
| `01-项目` | 当前进行的各类项目 |
| `02-领域` | 长期关注的责任领域（工作/学习/健康/财务/人际关系） |
| `03-资源` | 读书笔记、文章收藏、工具软件、灵感碎片 |
| `04-归档` | 已完成或废弃的内容 |
| `99-日记` | 个人日记与备忘录 |
| `模板` | 笔记模板文件 |

## 快速开始

1. 打开 Obsidian → "打开本地仓库" → 选择本文件夹
2. 设置 → 文件与链接 → 新建笔记存放位置：`00-收件箱`
3. 安装推荐插件：Dataview、Templater、QuickAdd
4. 开始使用模板创建笔记
